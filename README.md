# ✈️ Travel Booking - Plataforma de Agendamento de Viagens  

Este é um projeto desenvolvido com **React**, **TypeScript** e **Tailwind CSS**, integrando-se com um backend para fornecer uma experiência completa de agendamento de viagens. O projeto foi criado durante o evento **NLW Journey** da Rocketseat.  

## 🚀 Tecnologias Utilizadas  

- **React** + **TypeScript** - Construção da interface  
- **Tailwind CSS** - Estilização responsiva  
- **Axios** - Comunicação com o backend  

## 🎯 Funcionalidades  

✅ Pesquisa e agendamento de viagens  
✅ Interface responsiva e moderna   
✅ Integração com backend para gerenciamento de reservas  

✨ Aprendizados
Melhoria na organização de componentes no React
Uso dos principais hooks do React
Uso avançado de Tailwind CSS para estilização ágil
